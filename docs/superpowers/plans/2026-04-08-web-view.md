# WUPHF Web View Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `--web` flag to the WUPHF CLI that starts the broker + serves a web UI at localhost:7891, giving users a browser-based office that talks to the same broker API the TUI uses.

**Architecture:** The broker already runs on :7890 with a full REST API (messages, channels, members, tasks, requests, interviews). We add a thin HTTP file server on :7891 that serves `web/` static files. The web frontend polls the broker API for live data. A `--web` flag on the CLI starts broker + web server + agents (no tmux). The broker gets a `/web-token` endpoint so the web UI can authenticate.

**Tech Stack:** Go (net/http for static file serving), vanilla JS (no build step), broker REST API (already exists)

```go
func runWeb(args []string, packSlug string, unsafe bool, webPort int) {
    l, err := team.NewLauncher(packSlug)
    if err != nil {
        fmt.Fprintf(os.Stderr, "error: %v\n", err)
        os.Exit(1)
    }

    }
    if err := l.Preflight(); err != nil {
        fmt.Fprintf(os.Stderr, "error: %v\n", err)
        os.Exit(1)

    }
}
```

```go
func (l *Launcher) LaunchWeb(webPort int) error {
    mcpConfig, err := l.ensureMCPConfig()
    if err != nil {
        return fmt.Errorf("prepare mcp config: %w", err)
    }
    l.mcpConfig = mcpConfig
    killStaleBroker()
    l.broker = NewBroker()
    if err := l.broker.Start(); err != nil {
        return fmt.Errorf("start broker: %w", err)
    }
    l.broker.ServeWebUI(webPort)
    l.spawnBackgroundAgents()
    go l.notifyAgentsLoop()
    go l.notifyTaskActionsLoop()
    go l.pollNexNotificationsLoop()
    go l.watchdogSchedulerLoop()
    fmt.Printf("\n  Web UI:  http://localhost:%d\n", webPort)
    fmt.Printf("  Broker:  http://localhost:%d\n", BrokerPort)
    fmt.Printf("  Press Ctrl+C to stop.\n\n")
    select {}
}
```

- [ ] **Step 4: Add `ServeWebUI` to Broker**

In `internal/team/broker.go`:

```go
func (b *Broker) ServeWebUI(port int) {
    exePath, _ := os.Executable()
    webDir := filepath.Join(filepath.Dir(exePath), "web")
    if _, err := os.Stat(webDir); os.IsNotExist(err) {
        webDir = "web"
    }
    mux := http.NewServeMux()
    mux.Handle("/", http.FileServer(http.Dir(webDir)))
    go http.ListenAndServe(fmt.Sprintf("127.0.0.1:%d", port), mux)
}
```

- [ ] **Step 5: Add CORS middleware to broker**

Wrap the broker mux in `StartOnPort` so the web UI on :7891 can call :7890:

```go
func corsMiddleware(next http.Handler) http.Handler {
    return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
        w.Header().Set("Access-Control-Allow-Origin", "*")
        w.Header().Set("Access-Control-Allow-Headers", "Authorization, Content-Type")
        w.Header().Set("Access-Control-Allow-Methods", "GET, POST, PUT, DELETE, OPTIONS")
        if r.Method == "OPTIONS" {
            w.WriteHeader(http.StatusNoContent)
            return
        }
        next.ServeHTTP(w, r)
    })
}
```

- [ ] **Step 6: Add `/web-token` endpoint (localhost-only, no auth)**

```go
mux.HandleFunc("/web-token", b.handleWebToken)

func (b *Broker) handleWebToken(w http.ResponseWriter, r *http.Request) {
    host, _, _ := net.SplitHostPort(r.RemoteAddr)
    if host != "127.0.0.1" && host != "::1" {
        http.Error(w, "forbidden", http.StatusForbidden)
        return
    }
    w.Header().Set("Content-Type", "application/json")
    json.NewEncoder(w).Encode(map[string]string{"token": b.token})
}
```

### Task 2: Add broker API client to web frontend

**Files:**
- Modify: `web/index.html`

- [ ] **Step 1: Add WuphfAPI object**

Add at top of `<script>` section. All methods use `fetch()` with Bearer token auth. Convenience methods for `getMessages`, `postMessage`, `getMembers`, `getChannels`, `getTasks`, `getRequests`, `answerRequest`. Token fetched from `/web-token` on init.

Key methods:
- `init()` - fetches token from broker
- `get(path, params)` - GET with auth header
- `post(path, body)` - POST with auth header
- `getMessages(channel, sinceId, limit)` - polls `/messages` with `viewer_slug=human`
- `postMessage(content, channel)` - POST to `/messages` with `from: "you"`
- `getMembers()` - GET `/office-members`
- `getChannels()` - GET `/channels`
- `getRequests()` - GET `/requests`
- `answerRequest(id, choiceId)` - POST `/requests/answer`
